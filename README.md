# Gas Spending Tracker

A simple mobile-friendly web app to track gas fill-ups, price, station, and totals.

## How to use

1. Open `gas-tracker/index.html` in a browser.
2. Add your fill-up amount, price, currency, station, and date.
3. The app saves entries locally in the browser.
4. Use `Export CSV` to download your data.

## Notes

- Works offline once the page is loaded.
- You can install the app to your phone if the browser supports it.
- Data is stored in `localStorage` for the current browser.
